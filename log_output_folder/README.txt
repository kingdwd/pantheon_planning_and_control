Data stored following:

p = vehicle position
q = vehicle orientation
rc = roll commanded
pc = pitch commanded
yrc = yaw rate commanded
tc = thrust commanded
d = desired position
dyn = dynamic obstacle
vo1 = vertical obstacle 1
vo2 = vertical obstacle 2
ho1 = horizontal obstacle 1
t = target
k = camera intrinsics
iters = iterations needed to find the approximate solution
solve_time = time to find the approximate solution (microseconds)
